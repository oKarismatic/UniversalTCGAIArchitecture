CORE IDEA

A universal framework for analyzing, teaching, and understanding any card game by breaking gameplay into functional layers.

Not a solver or predictor — a structured system for decks, gameplay, and design understanding.

CORE LAYERS (STRUCTURAL + ADVANTAGE CHAIN)
R (Rules)

Game constraints:

win conditions
turn structure
timing/priority/stack rules
deck construction rules
resource system rules
A (Accessibility)

What can legally and economically be played:

costs
requirements
restrictions
setup constraints
