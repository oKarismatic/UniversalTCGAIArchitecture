# UniversalTCGAIArchitecture
Modular AI framework for TCG logic, featuring UROE v2 resolution engine, stress-proof edge-case handling, and dual-personality coaching agents.
My brother and I built this system because we were tired of AI tools that struggled with complex card game logic, timing, and rules. We wanted something that could actually "think" through a game state rather than just guessing
